## üöÄ What is This?

**Elevator Pitch Platform** is a revolutionary job search tool that lets you create, share, and track professional video elevator pitches. Instead of sending boring PDF resumes that get lost in the pile, send recruiters an engaging video that showcases your personality, communication skills, and passion.

### üéØ **For Job Seekers**
- Create compelling 90-second video elevator pitches
- Embed interactive buttons directly in your resume PDFs
- Track who's viewing your pitches with detailed analytics
- Stand out from the competition with professional video content

### üëî **For Recruiters**
- Get a complete picture of candidates before interviews
- See communication skills, personality, and presentation abilities
- Access candidate portfolios and LinkedIn profiles instantly
- Save time with pre-screened, video-qualified candidates

## üöÄ Quick Start

### Prerequisites
- Node.js 16+ 
- MongoDB
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bashirbobboi/elevator-pitch.git
   cd elevator-pitch
   ```

2. **Install dependencies**
   ```bash
   # Backend
   cd backend
   npm install
   
   # Frontend
   cd ../frontend
   npm install
   ```

3. **Set up environment**
   ```bash
   # Backend
   cd backend
   cp .env.example .env
   # Edit .env with your MongoDB connection
   ```

4. **Start the application**
   ```bash
   # Terminal 1 - Backend
   cd backend
   npm start
   
   # Terminal 2 - Frontend
   cd frontend
   npm run dev
   ```

5. **Access the application**
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:5001

## üì± How It Works

### **For Job Seekers**

1. **Create Your Pitch**
   - Write a compelling 90-second script
   - Record with professional teleprompter
   - Upload your resume as PDF

2. **Share with Recruiters**

   - Send to hiring managers
   - Track who's viewing your pitch

3. **Stand Out**
   - Show personality and communication skills

### **For Recruiters**

1. **Receive Pitch**
   - Get a professional link to candidate's pitch
   - View candidate profile and resume
   - Watch video elevator pitch

2. **Evaluate Candidate**
   - See communication skills in action
   - Assess personality and cultural fit
   - Access portfolio and LinkedIn

3. **Make Decisions**
